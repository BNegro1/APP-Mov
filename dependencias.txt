---------------------------------------------------------------------------------------


1. Instalar dependencias del proyecto:
   npm install

2. Instalar SQLite y plugins necesarios:
   cd discoverify
   npm install cordova-sqlite-storage
   npm install @awesome-cordova-plugins/sqlite

3. Instalar Toast visual:
   npm install @ionic-native/toast
   npm install cordova-plugin-x-toast

4. Instalar Capacitor para Android:
   npm install @capacitor/android
   npm i -g native-run

5. Agregar la plataforma Android al proyecto:
   npx cap add android

6. Ejecutar el proyecto en Android Studio:
   npx cap open android

7. Build y generaci√≥n de APK:
   - Compilar el proyecto Ionic:
     ionic build
   - Sincronizar Capacitor:
     npx cap sync android
   - Abrir en Android Studio:
     npx cap open android

8. Actualizar APK creada:
   ionic build
   npx cap sync android
   npx cap open android

9. En caso de problemas al build (para regenerar el entorno Android):
    Remove-Item -Recurse -Force ./android
    npx cap add android
    ionic build
    npx cap sync android
    npx cap open android
